# 📧 AI Email Generator

AI Email Generator built using LangChain + Streamlit + OpenRouter.

## 🚀 Features

- Generate professional emails
- Multiple tones:
  - Formal
  - Casual
  - Friendly
- Uses free LLMs from OpenRouter
- Clean Streamlit UI

## 🛠 Tech Stack

- Python
- Streamlit
- LangChain
- OpenRouter API

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run Project

```bash
streamlit run app.py
```

## 🔑 Add API Key

Create `.env` file:

```env
OPENROUTER_API_KEY=your_api_key
```
